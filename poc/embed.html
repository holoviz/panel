Embedding Panel Apps

This page demonstrates how Panel apps can be embedded in any HTML page with minimal boilerplate. Each approach uses a shared Pyodide runtime via panel_runner.html in an iframe.

1. Simple App Embed (No Editor)

Embed a Panel app by providing Python code. The iframe loads Pyodide once and renders the app. Users interact with the running app but cannot edit the code.

Embedding is as simple as:

<div class="panel-embed">
  <iframe id="app1" src="panel_runner.html"></iframe>
</div>
<script>
  panelEmbed('app1', `import panel as pn
pn.Column("# Hello!", "Panel runs in the browser!").servable()`);
</script>

2. Inline Editor + App

A compact playground widget: editable code on top, live app below. Users can modify the code and click Run to see changes.

3. Auto-Servable (Just Return a Value)

If the code doesn't call .servable(), the playground automatically wraps the last expression with pn.panel(). This makes embedding even simpler for quick demos.

4. The Dream: Minimal Boilerplate

The ultimate goal is a web component or simple function that makes embedding trivial. Here's the vision for the API: 

<!-- Future: Custom element approach -->
<panel-app>
import panel as pn
pn.Column("# It just works!", "No boilerplate needed.").servable()
</panel-app>

<!-- Future: Script tag approach -->
<script type="panel">
import panel as pn
slider = pn.widgets.FloatSlider(name="Value", start=0, end=10)
pn.Column(slider, pn.bind(lambda v: f"Value: {v}", slider)).servable()
</script>

For sites like Claude.ai artifacts (which only support JS/React), the embed can be done with a single iframe: 

<!-- Works in any context that allows iframes -->
<iframe
  src="https://your-cdn.com/panel_runner.html"
  style="width:100%;height:400px;border:none"
></iframe>
<script>
  // Send code to the iframe
  const iframe = document.querySelector('iframe');
  iframe.onload = () => {
    iframe.contentWindow.postMessage({
      type: 'run',
      code: 'import panel as pn\npn.Column("Hello from Claude!").servable()'
    }, '*');
  };
</script>
