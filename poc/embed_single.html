Embedding Panel Apps (Single Pyodide)

This page demonstrates the same demos as embed.html, but uses a single Pyodide instance instead of 3 separate iframes. This reduces memory from ~1.5GB to ~300-500MB, avoiding browser crashes.

Each app renders into its own <div> container using Panel's write() / write_doc() with scoped DOM queries and isolated exec namespaces.

1. Simple App Embed (No Editor)

Embed a Panel app by providing Python code. The single Pyodide runtime renders the app into a target div. Users interact with the running app but cannot edit the code.
